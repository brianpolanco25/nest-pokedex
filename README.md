# Execute in development

1. Clone repository
2. Execute
```
 yarn install 
```` 
3.  Install Nest CLI
``` 
   npm i -g @nestjs/cli
```
4. Run mongodb
```
  docker-compose up -d
```

## Stack
  * Nest
  * MongoDB
  * Docker
